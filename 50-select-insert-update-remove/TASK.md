# Task - Basic Database Operations

## Goal

Practice writing examples for common database operations in a safe way.

## Instructions

Create a file named:

```txt
queries.md
```

Inside this file, write simple examples for these actions:

```txt
1. Get users list
2. Add one new user
3. Change one user's name
```

## Example Tables

Use these example table names:

```txt
users
products
orders
cart_items
```

## Requirements

Your practice file should include:

- one example for getting data
- one example for adding data
- one example for changing data
- a short note under each example explaining what it does

## Important Note

When changing data, always target one specific row using a condition such as an id.

## Expected Learning

After this task, you should understand how basic database operations are used in real project work.

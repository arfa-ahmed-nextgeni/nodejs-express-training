# Practice Task - Node.js Syntax

## Task 1

Create a TypeScript file and declare:

- a string variable
- a number variable
- a boolean variable

Print all values using `console.log()`.

## Task 2

Create a function that accepts:

- name
- role

Return a formatted string and print it.

## Task 3

Create an array of 3 technologies and print each value.

## Expected Understanding

After this topic, I should be able to:

- Write basic JavaScript/TypeScript syntax in Node.js
- Run a file using Node.js tooling
- Use variables, functions, arrays, and console output
- Understand that Node.js uses JavaScript syntax

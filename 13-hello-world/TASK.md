# Practice Task - Hello World

## Task 1

Create:

```txt
13-hello-world/src/index.ts
```

Create a Hello World Express server.

## Task 2

Add a script:

```json
"dev:hello": "tsx 13-hello-world/src/index.ts"
```

## Task 3

Run:

```bash
npm run dev:hello
```

Verify:

```txt
http://localhost:3003
```

returns:

```txt
Hello World
```

## Expected Understanding

After this topic, I should be able to:

- Create a simple Express route
- Return a response using `res.send()`
- Run a basic Express server

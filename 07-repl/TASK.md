# Practice Task - REPL

## Task 1

Open Node.js REPL:

```bash
node
```

Run:

```js
2 + 3
```

## Task 2

Create a variable:

```js
const framework = "Express";
```

Print:

```js
framework
```

## Task 3

Create a function:

```js
function add(a, b) {
  return a + b;
}
```

Run:

```js
add(10, 20)
```

## Expected Understanding

After this topic, I should be able to:

- Explain what REPL means
- Open Node.js REPL
- Execute JavaScript directly in REPL
- Exit REPL
